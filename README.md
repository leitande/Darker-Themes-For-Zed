# Darker Themes For Zed

Custom darkened variants of popular Zed themes.

## Themes

### Rosé Pine Darker

A darkened version of the classic [Rosé Pine](https://github.com/rose-pine/zed) theme. The original `#191724` base background has been deepened to `#100e19` across all UI layers for a more subdued editing experience.

## Usage

1. Clone or download the theme `.json` files
2. Place them in `~/.config/zed/themes/`
3. Select the theme in Zed via `Ctrl+K T`
